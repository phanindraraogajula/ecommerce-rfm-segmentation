# 🛒 E-Commerce Customer Segmentation — RFM Analysis

## 📌 Project Overview
This project analyzes transactional e-commerce data to segment customers using the **RFM (Recency, Frequency, Monetary)** framework to identify Champions, At-Risk, and Lost customers.

## 🎯 Business Problem
E-commerce companies struggle to personalize marketing at scale. This analysis answers:
- Who are our most valuable customers?
- Which customers are at risk of leaving?
- How should we treat each segment differently?
